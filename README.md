# 📁 PyFile Manager

A sleek, interactive local file management utility built with Python and Streamlit. This application transitions a traditional CLI-based file handler into a modern visual dashboard.

👉 **[Live App Link](https://learning-python-from-nyc-0.streamlit.app/)**

## ✨ Features
* **Create Files:** Easily create text files with initial data injections.
* **Read Files:** Instantly preview and read local files with formatted code syntax.
* **Modify Files:** Append new strings, add new-line data, overwrite files completely, or safely rename files.
* **Delete Files:** Clean up files permanently with built-in UI warning safety blocks.

## 🛠️ Technology Stack
* **Language:** Python
* **UI Framework:** Streamlit
* **Core Library:** Pathlib (for robust file system management)
